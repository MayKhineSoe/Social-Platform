# Community Engagement Platform

A social-style web application built to demonstrate scalable frontend architecture and RESTful API design.

## Tech Stack
- Frontend: React, TypeScript, TanStack Query, Tailwind
- Backend: Node.js, Express, Prisma, PostgreSQL

## Features
- User authentication (JWT)
- Content feed with posts, likes, comments
- Follow / unfollow users
- RESTful API design
